# 파이썬(Python) 3.11.9 설치 완벽 가이드 (Windows 기준)

이 문서는 파이썬(Python)을 처음 접하시는 분들도 쉽게 따라 할 수 있도록 작성된 **Windows 환경 맞춤형 파이썬 3.11.9 버전 설치 가이드**입니다. 

---

## 1단계: 파이썬 공식 홈페이지 접속

1. 웹 브라우저를 열고 파이썬 공식 웹사이트인 [python.org](https://www.python.org/)에 접속합니다.
2. 상단 메뉴에서 **[Downloads]**에 마우스 커서를 올립니다.
3. 드롭다운 메뉴가 나타나면 **[Windows]**를 클릭합니다.

---

## 2단계: 파이썬 3.11.9 버전 다운로드

호환성 및 안정성이 검증된 **3.11.9 버전**을 설치하는 방법입니다.

1. Windows용 다운로드 페이지에서 스크롤을 내려 **Python 3.11.9** 버전을 릴리스 목록에서 찾거나, 단축키(`Ctrl + F`)를 눌러 `3.11.9`를 검색합니다.
2. 해당 버전의 다운로드 링크 중 시스템 환경에 맞는 설치 파일을 클릭합니다.
   * 대부분의 컴퓨터는 64비트이므로 **[Windows installer (64-bit)]**를 선택하여 다운로드합니다.
   * 👉 **[빠른 다운로드: Python 3.11.9 (64-bit) 다이렉트 링크 (클릭)](https://www.python.org/ftp/python/3.11.9/python-3.11.9-amd64.exe)**

---

## 3단계: 파이썬 설치 진행 (⭐가장 중요⭐)

다운로드한 설치 파일(`python-3.11.9-amd64.exe`)을 실행합니다. **여기서 가장 주의해야 할 설정이 있습니다.**

1. 첫 설치 화면이 나타나면, 창의 맨 아래에 있는 **`[✔] Add python.exe to PATH`** 항목을 **반드시 체크**합니다. (기본값은 해제되어 있으므로 직접 클릭해야 합니다.)
   > **💡 왜 체크해야 하나요?** 이 옵션을 체크해야 컴퓨터가 파이썬이 어디에 설치되어 있는지 인식할 수 있습니다. 나중에 명령 프롬프트(CMD)나 VS Code 같은 에디터에서 오류 없이 파이썬을 실행하기 위한 필수 과정입니다.
2. 체크를 완료했다면, 화면 상단의 **[Install Now]**를 클릭하여 설치를 시작합니다.
3. "Setup was successful" 이라는 메시지가 나타나면 설치가 무사히 완료된 것입니다. **[Close]**를 눌러 창을 닫습니다.

---

## 4단계: 설치 정상 완료 확인하기

마지막으로 파이썬이 정상적으로 설치되었는지 확인해 봅니다.

1. 키보드에서 `Windows 로고 키 + R`을 동시에 눌러 '실행' 창을 띄웁니다.
2. 빈칸에 `cmd`를 입력하고 **[확인]**을 눌러 명령 프롬프트를 엽니다.
3. 검은 화면이 열리면, 아래 명령어를 똑같이 입력하고 `Enter` 키를 누릅니다.
   ```bash
   python --version
   ```
4. 다음 줄에 `Python 3.11.9` 라고 출력된다면 완벽하게 설치가 끝난 것입니다! 🎉 즐거운 코딩 되세요!
