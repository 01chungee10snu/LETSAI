# [Prompt] KOTESTMkiwi: 한국어 텍스트 마이닝 및 다차원 분석 통합 플랫폼 개발

## 1. 프로젝트 개요 및 환경 설정
본 프로젝트는 연구소 및 실무 환경에서 한국어 텍스트 데이터를 전문적으로 분석하기 위한 **'KOTESTMkiwi'** 플랫폼 개발을 목적으로 합니다. 뉴스 기사, 설문 주관식 응답, 소셜 미디어 데이터 등 다양한 텍스트 소스로부터 깊이 있는 통계적·감성적 인사이트를 도출하는 '올인원 분석 솔루션' 구축이 핵심입니다.

*   **구동 환경:** 오프라인 윈도우(Windows) 환경 (인터넷 연결 불가)
*   **기술 스택:** Streamlit 기반 애플리케이션
*   **안정성 요구사항:** 외부 라이브러리 의존성(PyTorch 등) 및 DLL 충돌 방지를 위한 최적화된 구조

## 2. 지능형 분석 워크플로우 (Stateful Design)
사용자의 편의를 위해 단계별 진행 상황을 사이드바에 표시하고(✅ 완료, ▶️ 진행 중, ⏸️ 대기), 상태(State)를 유지하며 다음 단계로 이동하는 워크플로우를 구현합니다.

1.  **[시스템 진단]** 애플리케이션 시작 시 가용 메모리, 디스크 공간, GPU 가속 가능 여부를 확인하여 표시합니다.
2.  **[데이터 업로드]** CSV/Excel 파일 업로드 기능을 제공하며, 인코딩(UTF-8, CP949 등)을 자동으로 감지하여 데이터 무결성을 검증합니다.
3.  **[데이터 전처리 및 지정]** 분석 대상 텍스트 열과 통계적 공변인(성별, 지역, 직급 등)을 사용자가 직접 지정할 수 있어야 합니다.
4.  **[분석 파라미터 설정]** 분석 알고리즘(감성 분석, 토픽 모델링 등) 선택 및 하드웨어 자원 분배(배치 사이즈, 학습률 등) 옵션을 제공합니다.
5.  **[프로세싱 및 모니터링]** 분석 실행 시 백그라운드 작업을 실행하며, 실시간 프로그레스 바를 통해 진행률을 피드백합니다.
6.  **[결과 익스포트]** 분석 결과 데이터프레임 및 시각화 차트를 엑셀 보고서 또는 고해상도 이미지 파일로 일괄 저장합니다.

## 3. 핵심 분석 모듈 상세 기술 요구사항
분석 엔진은 최신 한국어 NLP 모델 및 통계 기법을 활용합니다.

*   **다차원 감성 분석 (Multidimensional Sentiment Analysis):**
    *   KOTE 모델을 활용하여 6종 이상의 세부 감정(기쁨, 슬픔, 분노 등)을 정량적으로 추출합니다.
    *   각 감정별 추출 신뢰도 점수를 함께 산출하여 데이터의 정합성을 보장합니다.
*   **구조적 토픽 모델링 (Structural Topic Modeling, STM):**
    *   사전에 정의된 공변인(Covariates)에 따른 토픽 분포(Prevalence) 및 내용(Content) 변화를 통계적으로 분석합니다.
    *   집단 간 토픽 전환 확률 및 유의미한 키워드 차이를 도출합니다.
*   **의미망 및 시각화 분석 (Network & Time-series):**
    *   감정-토픽-핵심 단어 간의 상관관계를 동적인 연결망 그래프(SNA)로 구현합니다.
    *   시간 흐름에 따른 감성 변화 추이를 확인할 수 있는 시계열 대시보드와 워드클라우드를 포함합니다.

## 4. 시스템 안정성 및 유틸리티 기능
*   **리소스 모니터링:** 시스템 자원 사용량을 실시간으로 감시하는 대시보드 패널을 상시 노출합니다.
*   **설정 자동화 (Preset System):** 사용자가 설정한 분석 하이퍼파라미터 및 전처리 옵션을 프리셋으로 저장하고 재사용할 수 있게 합니다.
*   **장애 복구 및 로그:** 상세 실행 로그를 기록하는 전용 뷰어를 구현하고, 분석 중단 시 최종 체크포인트부터 재개 가능한 복구 기능을 포함합니다.

## 5. 구현 가이드라인
본 플랫폼은 향후 기능 확장을 고려하여 **모듈화된 코드 구조(Modular Architecture)**로 작성되어야 합니다. 가독성 높은 주석과 객체 지향적 설계를 통해 유지보수가 용이하도록 제작해 주시기 바랍니다.
